## 🧮 الخوارزمية الذكية

### المنطق الأساسي:

```typescript
if (لا توجد فترات سابقة) {
  → إنشاء من 1-15 من الشهر الحالي
} else {
  آخر_يوم = آخر_تاريخ_في_البيانات
  
  if (آخر_يوم == 15) {
    → إنشاء من 16 إلى نهاية الشهر (28/30/31)
  } else {
    → الانتقال للشهر القادم
    → إنشاء من 1-15
  }
}
```

### أمثلة الحسابات:

**مثال 1: من 1-15 إلى 16-31**
```
Input:  آخر تاريخ = 15/3/2026
Output: الفترة التالية = 16/3 إلى 31/3

حساب الأيام:
- اليوم 1: 16/3/2026
- اليوم 2: 17/3/2026
- ...
- اليوم 14: 29/3/2026 (النصف الثاني من النصف الثاني = 14 يوم)
```

**مثال 2: من نهاية الشهر إلى الشهر الجديد**
```
Input:  آخر تاريخ = 31/3/2026 (أو 28/2، 30/4، إلخ)
Output: الفترة التالية = 1/4 إلى 15/4

حساب الأيام:
- اليوم 1: 1/4/2026
- اليوم 2: 2/4/2026
- ...
- اليوم 14: 14/4/2026
```

## 🔧 واجهة البرمجة (API)

### دالة `getNextPeriod(lastEndDateStr: string): PeriodInfo`

**الإدخال:**
```typescript
lastEndDateStr: "2026-03-15"  // آخر تاريخ في الفترة
```

**الإخراج:**
```typescript
{
  startDate: Date,          // كائن التاريخ
  endDate: Date,            // كائن التاريخ
  startDateStr: string,     // "2026-03-16"
  endDateStr: string,       // "2026-03-31"
  startDay: number,         // 16
  endDay: number,           // 31
  month: number,            // 3
  year: number,             // 2026
  label: string             // "من 16 إلى 31/3/2026"
}
```

---

### دالة `generateDates(startDateStr: string): string[]`

**الإدخال:**
```typescript
startDateStr: "2026-03-01"  // يوم البداية
```

**الإخراج:**
```typescript
[
  "2026-03-01",
  "2026-03-02",
  // ...
  "2026-03-14"  // 14 يوم دقيق
]
```

## 💾 هيكل البيانات

### Entry (الإدخال)
```typescript
interface Entry {
  id: string;           // معرف فريد
  date: string;         // "2026-03-01"
  revenue: number;      // إيداع
  coupons: number;      // بونات
  debitNote: string;    // مذكرة خطأ مدين
  invoices: number;     // فواتير
  creditNote: string;   // مذكرة خطأ دائن
}
```

### PeriodInfo (معلومات الفترة)
```typescript
interface PeriodInfo {
  startDate: Date;      // أول يوم
  endDate: Date;        // آخر يوم
  startDateStr: string; // ISO format
  endDateStr: string;   // ISO format
  startDay: number;     // رقم اليوم (1 أو 16)
  endDay: number;       // رقم آخر يوم
  month: number;        // رقم الشهر
  year: number;         // السنة
  label: string;        // نص توضيحي
}
```

## 🔐 معالجة الأخطاء

### الأخطاء المحتملة والحلول:

| الخطأ | السبب | الحل |
|------|------|------|
| تاريخ غير صحيح | صيغة خاطئة | التحقق من ISO format |

| تداخل الفترات | الحساب الذكي مخطئ | تحقق من منطق getNextPeriod |
